# E-commerce Sales Analysis

## Objective

Analyze e-commerce sales performance and identify key business insights.

## Tools

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## KPIs Analyzed

* Total Revenue
* Top Products
* Monthly Sales Trends
* Customer Segmentation

## Key Insights

* December was the highest-performing month
* Top 20% products generated 80% of revenue
* Premium customers drove most sales
